# V.E.W Contracting Website

Custom construction company website and branding project built using HTML and CSS.

## Overview

This project was created to develop a professional online presence for V.E.W Contracting. The website focuses on clean branding, responsive design, and a modern layout tailored for a construction business.

## Features

- Responsive website layout
- Custom company branding
- Modern homepage design
- Navigation bar
- Hero section with company imagery
- Mobile-friendly structure

## Technologies Used

- HTML
- CSS
- Visual Studio Code
- GitHub

## Project Goals

- Improve business credibility
- Create a strong online presence
- Showcase construction services professionally
- Develop a scalable website structure

## Current Progress

This project is currently in development and will continue to be updated with:
- Additional pages
- Improved responsiveness
- Enhanced UI/UX
- JavaScript functionality
- Portfolio and contact sections
